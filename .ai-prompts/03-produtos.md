# Tarefa: Criar Grid de Produtos

Crie um componente de vitrine de produtos.
- Layout: Grid CSS (1 coluna mobile, 2 tablet, 4 desktop).
- Card de Produto:
  - Fundo branco, borda preta, sombra dura.
  - Imagem do produto no topo.
  - Título do produto em negrito.
  - Preço riscado (se houver desconto) e preço atual.
  - Botão de "Adicionar" (completamente largo ocupando a base do card).
- Crie um array de dados mockados com 4 produtos falsos para testar a renderização.
